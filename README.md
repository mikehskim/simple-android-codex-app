# Hello World Compose Android App

Minimal single-screen Android app built with Kotlin + Jetpack Compose.

## Requirements implemented

- One screen
- Displays `Hello World`
- Gradle Kotlin DSL (`*.kts`)
- Minimum SDK 24

## Project structure

- `app/src/main/java/com/example/helloworldcompose/MainActivity.kt`: Compose UI and activity entry point
- `app/build.gradle.kts`: Android and Compose module configuration
- `settings.gradle.kts`, `build.gradle.kts`: root Gradle KTS configuration
